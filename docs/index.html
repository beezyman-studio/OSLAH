Getting Started

Set up OSLAH and connect your team in under 5 minutes.

1

Install Ollama Locally

Download and install Ollama on the host machine. Run your chosen models locally (e.g., ollama run deepseek-r1:7b).

2

Run OSLAH Installer

Download and execute OSLAH_Setup.exe. Launch the OSLAH desktop application from your Windows workspace.

3

Configure and Share Server API

Enter your administrative API keys, launch the local network server dashboard, and distribute your endpoint across the local subnet.
